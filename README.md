# Syncthing Service

Syncthing is a service to syncronize files between two folders remotely or locally.

For more information about Syncthing click [here][Syncthing site].

## How to install and run

- First make a copy of the ".env.example" file and rename it to ".env";
- Fill the required variables in the .env file;

After that you can run the command:

`docker-compose up -d`

[Syncthing site]: https://syncthing.net
